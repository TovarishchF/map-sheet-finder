# Советская разграфка

Интерактивный инструмент для работы с номенклатурой листов топографических карт, основанный на системе разграфки, принятой в СССР.

**[Открыть приложение](https://tovarishchf.github.io/map-sheet-finder/)**

## Функциональные возможности

- Определение географических границ листа по его номенклатурному обозначению (например, `N-37-56-А-в-1`).
- Поиск листа миллионной разграфки по заданным координатам.
- Последовательный переход между масштабами от 1:1 000 000 до 1:2 000 с соблюдением официальной системы деления.
- Навигация по истории просмотренных листов.
- Корректная обработка листов южного полушария (суффикс `(Ю.П.)`).
- Поддержка сдвоенных и счетверённых листов для широт свыше 60° и 76° соответственно.
- Экспорт границ листов в формате GeoJSON в системе координат EPSG:4326.  

## Инструкция по использованию

1. Введите номенклатурное обозначение в строку поиска и нажмите «Найти».
2. Для детализации масштаба выберите лист сетки щелчком мыши.
3. Воспользуйтесь панелью «Выберите следующий масштаб» для перехода к более крупному масштабу.
4. Для определения листа по координатам заполните поля широты и долготы и нажмите «Перейти».

## Техническая реализация

- Картографическая основа OpenStreetMap — библиотека [Leaflet](https://leafletjs.com/).
- Интерфейс построен на HTML5, CSS3 и нативном JavaScript (ES6).
- Логика разграфки реализована в соответствии с нормативными документами ГУГК СССР.

## Адаптация к мобильным устройствам

Приложение адаптировано для работы на устройствах с малыми экранами: элементы управления перестраиваются, боковая информационная панель смещается под область карты.
